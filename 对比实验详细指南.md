### 1.2 核心概念

| 组件 | 说明 | 输入 | 输出 |
|------|------|------|------|
| **VAE** | 变分自编码器，学习抓取数据的隐空间 | 点云 (1024) + 抓取 (6D) | 隐向量 (4D) |
| **Diffusion** | 反向扩散过程，从高斯噪声逐步去噪生成 | 条件信息 (点云特征) + 噪声 | 抓取隐向量 |
| **Flow Matching** | 连续流匹配，直接学习从噪声到数据的轨迹 | 条件信息 + 起点噪声 | 抓取隐向量 |

### 1.3 实验目标

- **验证两种生成方式的有效性**：Diffusion vs Flow Matching
- **对比生成质量**：成功率、多样性
- **对比生成速度**：推理时间
- **分析对称性**：VAE 隐空间的利用效率

### 3.5 VAE 训练常见问题

#### 问题 1：假训练（global_step 不涨）

**现象**：
- Epoch 快速跳过，但 `global_step` 卡在某个值
- Loss 突然飙升到 0.75+
- 步速异常快（>7 it/s）

**解决方案**：
```bash
# 1. 终止训练
pkill -f train_generator.py

# 2. 清空旧 checkpoint
rm -rf output/comparison/exp_diffusion_vs_fm/vae/*

# 3. 修改配置文件设 resume_training_from_last = False

# 4. 重新启动训练
```

#### 问题 2：数据加载卡死

**现象**：
- 训练启动后停在 "Sanity Checking" 或 "Epoch 0: 0%"
- GPU 利用率为 0%
- 无日志输出

**解决方案**：
```python
# 在配置文件中设置
num_workers_per_gpu = 0  # 禁用多线程数据加载

# 重新启动训练
```

#### 问题 3：CUDA Out of Memory

**现象**：
- 训练中报错 `CUDA out of memory`
- GPU 显存被占满

**解决方案**：
```python
# 在配置文件中降低 batch_size
batch_size = 16  # 从 32 降至 16

# 或减少 num_workers
num_workers_per_gpu = 0
```

```python
    type="TimeConditionedResNet1D",
    args=dict(
        dim=grasp_latent_dims,           # 4
        channels=1,
        block_channels=(32, 64, 128, 256),

        dropout=0.1,

)

    ddm=dict(
        model=dict(
            type="GraspLatentDDM",
            args=dict(
                model=denoiser_model,
                latent_in_features=grasp_latent_dims,  # 4

                noise_scheduler_type="ddpm",
                diffusion_loss="l2",
                beta_schedule="linear",
                is_conditioned=True,
                joint_training=False,  # 不联合训练 VAE
                denoising_loss_weight=1.0,
                variance_type="fixed_large",
                elucidated_diffusion=False,
                beta_start=0.00005,
                beta_end=0.001,
```

## 结果对比与评估

### 7.1 定量评估指标

#### 指标定义

| 指标 | 说明 | 计算方法 |
|------|------|--------|
| **Success Rate** | 成功抓取的比例 | 模拟验证 / 总生成数 |
| **Diversity** | 生成抓取的多样性 | 隐空间中样本间的平均距离 |
| **Inference Speed** | 推理速度 | 平均每个样本的推理时间 (ms) |
| **Reconstruction Loss** | VAE 重构误差 | MSE(原始, 重构) |
| **Coverage** | 覆盖率 | 生成样本覆盖的隐空间体积 |

### 7.2 生成评估脚本

创建文件 `tools/evaluate_comparison.py`：

```python
import torch
import numpy as np
from pathlib import Path
from tools.inference import InferenceVAE, InferenceLDM

def evaluate_models(
    vae_path="output/comparison/exp_diffusion_vs_fm/vae",
    ddm_path="output/comparison/exp_diffusion_vs_fm/ddm",
    num_samples=100,
    device="cuda:0"
):
    """对比评估 VAE 和 Diffusion 模型"""
    
    results = {}
    
    # 1. 加载模型
    inference_vae = InferenceVAE(exp_path=vae_path, device=device)
    inference_ddm = InferenceLDM(exp_path=ddm_path, device=device)
    
    # 2. 评估 VAE 重构能力
    print("评估 VAE 重构能力...")
    recon_losses = []
    for i in range(num_samples):
        pc = torch.randn(1, 1024, 3).to(device)
        grasp = torch.randn(1, 6).to(device)
        
        with torch.no_grad():
            z_mean, z_logvar = inference_vae(pc, grasp)
            z = z_mean  # 取均值
            recon_grasp = inference_vae.decode(z)
            loss = torch.nn.functional.mse_loss(recon_grasp, grasp).item()
            recon_losses.append(loss)
    
    results['vae_recon_loss'] = np.mean(recon_losses)
    print(f"✓ VAE 平均重构误差: {results['vae_recon_loss']:.4f}")
    
    # 3. 评估 Diffusion 生成速度
    print("评估 Diffusion 生成速度...")
    import time
    pc = torch.randn(1, 1024, 3).to(device)
    
    times = []
    for i in range(10):
        start = time.time()
        with torch.no_grad():
            grasps = inference_ddm(pc, num_samples=1)
        times.append(time.time() - start)
    
    results['ddm_inference_time'] = np.mean(times)
    print(f"✓ Diffusion 平均推理时间: {results['ddm_inference_time']:.4f}s")
    
    # 4. 多样性评估
    print("评估生成多样性...")
    with torch.no_grad():
        # VAE 采样多样性
        vae_samples = []
        for _ in range(num_samples):
            z = torch.randn(1, 4).to(device)  # 从隐空间采样
            sample = inference_vae.decode(z)
            vae_samples.append(sample)
        vae_samples = torch.cat(vae_samples, dim=0)  # (100, 4)
        
        # 计算成对距离
        vae_diversity = torch.pdist(vae_samples).mean().item()
        print(f"✓ VAE 采样多样性 (平均距离): {vae_diversity:.4f}")
        results['vae_diversity'] = vae_diversity
        
        # Diffusion 采样多样性
        ddm_samples = []
        for i in range(num_samples):
            pc = torch.randn(1, 1024, 3).to(device)
            sample = inference_ddm(pc, num_samples=1)
            ddm_samples.append(sample)
        ddm_samples = torch.cat(ddm_samples, dim=0)  # (100, 4)
        
        ddm_diversity = torch.pdist(ddm_samples).mean().item()
        print(f"✓ Diffusion 采样多样性: {ddm_diversity:.4f}")
        results['ddm_diversity'] = ddm_diversity
    
    return results

if __name__ == "__main__":
    results = evaluate_models()
    
    print("\n" + "="*50)
    print("对比评估结果总结")
    print("="*50)
    for key, value in results.items():
        print(f"{key:30s}: {value:.6f}")
```

```python
def visualize_latent_space(

    ddm_z = []
```
